> 运行 mongoexport -h localhost:27017 -u 'db_name' -p 'password' -d [db] -c [collection] -o a.json 可导出数据库集合到 json
>
> MongoDB 没有表的概念，无法导出表结构，只能导出全量 JSON 数据
>
> 现已导出 user.json （用户信息，便于权限管理）
>
> MongoDB 支持动态创建集合，在插入数据时，若不存在集合，则创建。
